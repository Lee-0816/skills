# Product Research Skill

> Verdent skill: 从小红书、知乎、什么值得买等平台自动采集产品评测信息，生成结构化的选购攻略。

## 它能做什么

输入一句话，自动完成从数据采集到攻略生成的全流程：

```
你: "帮我调研小红书上关于新生儿奶粉的推荐，生成一个新手奶爸的奶粉攻略"
```

Skill 将自动执行：

1. 生成 4-6 个多角度搜索关键词
2. 通过 Playwriter 控制真实浏览器，批量搜索小红书并拦截 API 获取 200+ 条笔记数据
3. 从知乎、什么值得买等平台补充专业评测和参数数据
4. 去重排序 → 品牌频次分析 → 参数对比 → 编译为结构化 Markdown 攻略

**输出示例**（节选）：

```markdown
## 二、品牌推荐（按小红书热度排名）

| 品牌 | 亮点 | 参考价格 | 来源 |
|------|------|---------|------|
| [爱他美卓萃](https://www.xiaohongshu.com/explore/xxx) | HMO + OPN 活性蛋白 | ~310元/800g | [老勋测评](https://www.xiaohongshu.com/explore/xxx) |
| ... | ... | ... | ... |

## 九、参考来源
### 小红书笔记
| # | 笔记标题 | 作者 | 点赞 | 链接 |
|---|---------|------|------|------|
| 1 | 九大奶粉品牌天花板 | 老勋 | 7.6万 | [查看](https://www.xiaohongshu.com/explore/xxx) |
```

## 前置条件

1. **Verdent** — AI coding assistant（[下载](https://verdent.dev)）
2. **Playwriter skill** — 用于驱动 Chrome 访问小红书
3. **Playwriter Chrome 扩展** — [Chrome Web Store 安装](https://chromewebstore.google.com/detail/playwriter-mcp/jfeammnjpkecdekppnclgkkfahahnfhe)
4. **Node.js** >= 18（需要 `npx` 命令可用）

## 安装

### 方式一：克隆到 Verdent skills 目录（推荐）

```bash
git clone https://github.com/Lee-0816/product-research-skill.git ~/.verdent/skills/product-research
```

### 方式二：手动复制

将本仓库所有文件复制到 `~/.verdent/skills/product-research/` 目录下即可。

### 验证安装

在 Verdent 中输入 `@product-research`，如果能成功加载 skill 提示则安装成功。

## 使用方法

### 基本用法

在 Verdent 对话中输入：

```
@product-research 帮我调研小红书上关于 [你的主题] 的推荐，生成一份选购攻略
```

### 使用示例

```
# 母婴类
@product-research 调研新生儿奶粉推荐，生成新手奶爸攻略

# 数码类
@product-research 调研2025年降噪耳机推荐，生成一份千元价位选购指南

# 户外类
@product-research 调研露营装备推荐，生成新手露营入门攻略

# 家居类
@product-research 调研扫地机器人评测，生成家用扫地机选购攻略

# 美妆类
@product-research 调研油皮护肤品推荐，整理一份控油护肤攻略
```

### 首次使用注意

1. 确保 Chrome 浏览器已打开
2. 点击 Playwriter 扩展图标使其变绿（表示连接成功）
3. Skill 运行时会自动打开小红书搜索页面，通过 API 拦截采集数据（无需登录）

## 工作流程

```
用户输入主题
    │
    ├─ Phase 1: 关键词规划
    │    └─ 从推荐、教程、品牌对比、参数、避坑等角度生成 4-6 个搜索词
    │
    ├─ Phase 2: 小红书数据采集 (Playwriter)
    │    ├─ 创建浏览器会话
    │    ├─ 设置 API 响应拦截器
    │    ├─ 批量搜索关键词，每个关键词采集 20-40 条笔记
    │    └─ 提取: 标题、作者、点赞、收藏、标签、笔记链接
    │
    ├─ Phase 3: 多平台补充调研
    │    ├─ 知乎: 深度评测文章
    │    ├─ 什么值得买: 产品参数、价格追踪
    │    ├─ 官方产品页: 验证规格和定价
    │    └─ 记录每条数据的来源 URL
    │
    └─ Phase 4: 分析与生成
         ├─ 去重、按互动量排序
         ├─ 品牌/产品提及频次统计
         ├─ 多源交叉验证参数数据
         ├─ 在报告中内联参考链接
         └─ 输出结构化 Markdown 攻略（含参考来源汇总）
```

## 输出报告结构

生成的攻略包含以下章节：

| 章节 | 内容 |
|------|------|
| 基础知识 | 品类分类、核心选购指标、避坑要点 |
| 品牌/产品推荐 | 按热度分梯队推荐，附参考链接 |
| 核心参数对比表 | 多品牌参数横评，标注数据来源 |
| 按需求速选指南 | 需求 → 推荐产品 → 理由映射表 |
| 实用教程 | 基于高赞笔记整理的操作指南 |
| 常见问题与避坑 | FAQ，附来源链接 |
| 购买渠道与省钱建议 | 渠道对比、优惠策略 |
| 推荐关注账号 | 优质博主，附主页/代表笔记链接 |
| 参考来源 | 小红书笔记 + 其他平台文章汇总表 |

## 适用品类

本 skill 适用于任何消费品类的调研，已内置品类适配指南：

- **食品/母婴** — 成分对比表、安全标准、科普速查
- **数码/电子** — 参数对比、跑分数据、价格走势
- **户外/运动** — 场景匹配、装备清单、实测体验
- **美妆/护肤** — 肤质匹配、成分分析、使用顺序
- **家居/家电** — 空间适配、能效对比、安装注意事项

## 常见问题

### 小红书需要登录怎么办？

不需要登录。Skill 通过拦截浏览器 API 响应获取搜索数据，即使弹出登录框也能正常采集。遇到弹窗会自动按 Escape 关闭。

### Playwriter 扩展连接不上？

1. 确保 Chrome 浏览器已打开
2. 在任意网页上点击一次 Playwriter 扩展图标
3. 图标变绿即连接成功
4. 如果还是不行，尝试 `npx playwriter@latest session reset <id>`

### 采集的笔记数量不够？

- 每个关键词通常采集 20-40 条笔记
- 增加搜索关键词数量（从不同角度）可以提升总量
- 关键词之间保持 3 秒以上间隔，避免被限流

### 能采集笔记正文内容吗？

小红书 Web 版的笔记详情页需要登录才能查看，目前只能通过搜索 API 获取标题、描述和互动数据。如果需要正文内容，建议在 Chrome 中手动登录小红书后再使用。
