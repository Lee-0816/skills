## Adaptation guidelines

- **食品/母婴类**: 强调成分对比表、安全标准、科普速查
- **数码/电子类**: 强调参数对比、跑分数据、价格走势
- **户外/运动类**: 强调场景匹配、装备清单、实测体验
- **美妆/护肤类**: 强调肤质匹配、成分分析、使用顺序
- **家居/家电类**: 强调空间适配、能效对比、安装注意事项

## Quality checklist

- [ ] 数据来源清晰标注（采集日期、笔记数量、参考博主）
- [ ] 推荐有数据支撑（点赞数、收藏数）
- [ ] 成分/参数对比表完整且单位统一
- [ ] 包含速选指南（按需求→产品映射）
- [ ] 包含实用操作教程
- [ ] 包含避坑/FAQ 部分
- [ ] 价格信息标注为参考价
- [ ] **品牌推荐表中产品名附有参考链接**
- [ ] **参数/成分数据旁附有数据来源链接**
- [ ] **教程/攻略段落标注了参考笔记链接**
- [ ] **推荐博主附有代表笔记或主页链接**
- [ ] **末尾包含完整的"参考来源"汇总表**
- [ ] 所有链接格式正确可点击（Markdown `[text](url)` 格式）
- [ ] 末尾声明非广告、数据可追溯
